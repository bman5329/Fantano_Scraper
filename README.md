# theneedledrop review scraper for quick music discovery

## Getting Started
- Follow the steps for getting a youtube API key here https://theplusaddons.com/blog/how-to-get-youtube-api-key/
- Make a .env file and add YOUTUBE_APIE_KEY = "your_key_here"
- Make a virtual environment and install dependencies using requirements.txt
- run python3 app.py and go to http://127.0.0.1:5000
- Filter by timeframe, sort and go wild
